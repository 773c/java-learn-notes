##### 一、安装node.js

##### 二、安装webpack
> ---- npm install -g webpack@版本号 （vue-cli2依赖3.6.0）

##### 三、安装vue-cli脚手架
> ---- npm install -g @vue/cli

##### 四、拉取2.X版本的脚手架
> ---- npm install -g @vue/cli-init

##### 五、vue-cli初始化项目
（1）vue-cli2

> ---- vue init webpack 项目名称

（2）vue-cli3

> ---- vue create 项目名称

##### 六、axios安装
> ---- npm install axios --save（--save可以简写为-S）
