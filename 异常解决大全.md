* org.springframework.http.converter.HttpMessageNotWritableException: 
  No converter found for return value of type: class com.cjj.qlmusic.common.util.ResponseResultUtil

>  异常说明返回值有问题，检查下ResponseResultUtil类的有没有写get方法

* java.io.InvalidClassException: no valid constructor

> 一般有三种可能
>
> （1）存在抽象父类
>
> （2）抽象父类中存在有参数的构造方法
>
> （3）子类实现了序列化接口，父类没有实现
>
> 解决：
>
> 子类实现无参构造即可
